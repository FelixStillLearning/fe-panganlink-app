# fe-panganlink-app

React, Vite, and TypeScript frontend template.

## Requirements

- Node.js 20+
- npm 10+

## Setup

1. Copy environment variables: `cp .env.example .env`
2. Fill in values in `.env`
3. Install dependencies: `npm install`
4. Run: `npm run dev`

## Structure

Feature-based organization with generic folders for shared components, layout components, hooks, routes, API helpers, and example features.
