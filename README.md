## 🚀 Setup & Instalasi

### Persyaratan
- **Node.js** (Versi 20+)
- **npm** (Versi 10+)

### Langkah-langkah
1. Salin file environment:
   ```bash
   cp .env.example .env
   ```
2. Sesuaikan nilai variabel di `.env` jika diperlukan.
3. Instal semua dependensi:
   ```bash
   npm install
   ```
4. Jalankan *development server*:
   ```bash
   npm run dev
   ```
5. Akses aplikasi melalui browser pada `http://localhost:5173`.
